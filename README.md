📊 Pandas Basics – Streamlit App

This project is a simple and interactive Streamlit app that helps beginners learn the basics of the pandas library in Python.
The app demonstrates how to:

✔️ Read different file formats (CSV, JSON, Excel)

✔️ Create a DataFrame

✔️ View DataFrame summary using describe() and info()

✔️ Save DataFrames in multiple formats

✔️ Display a styled data preview using HTML

Perfect for students who are starting with data analysis in Python!

🚀 Features
📁 1. Read Files

Learn how to load different file types using:

pd.read_csv()

pd.read_json()

pd.read_excel()

🛠️ 2. Create DataFrame

Understand DataFrame creation using a Python dictionary.

📈 3. Describe & Info

Explore:

df.describe() → for numeric summary

df.info() → for structure and data types

💾 4. Save Files

Save your DataFrame to:

CSV

JSON

Excel

👀 5. Data Preview

A colorful HTML table preview is included to visualize the dataset.
